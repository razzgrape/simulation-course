# Лабораторная работа №5 — Моделирование случайных событий

## Описание задачи

Цель работы — реализовать программное моделирование случайных событий с использованием генератора псевдослучайных чисел. Реализованы два независимых приложения: генератор ответов «Да / Нет» и «Шар предсказаний» (Magic 8-Ball).

---

## Часть 1. Приложение «Да или Нет»

### Идея

Приложение моделирует простой бинарный случайный эксперимент: каждый вызов независимо возвращает один из двух исходов — «ДА» или «НЕТ» — с заданной вероятностью p.

### Реализация

```python
def get_yes_no(p=0.5):
    return "ДА" if random.random() < p else "НЕТ"
```

`random.random()` возвращает равномерно распределённое вещественное число из интервала [0, 1). Если оно меньше порога p — результат «ДА», иначе — «НЕТ». При p = 0.5 оба исхода равновероятны.

Параметр p настраивается пользователем в диапазоне [0, 1], что позволяет моделировать несимметричные события: например, при p = 0.8 ответ «ДА» будет выпадать в среднем в 80% случаев.

## Часть 2. Приложение «Шар предсказаний» (Magic 8-Ball)

### Идея

Классический шар предсказаний содержит 20 ответов: 10 положительных, 5 нейтральных и 5 отрицательных. Каждый ответ равновероятен — вероятность выпадения любого конкретного ответа составляет 1/20 = 0.05.

### Список ответов

```python
MAGIC_ANSWERS = [
    "Бесспорно", "Предрешено", "Никаких сомнений",
    "Определённо да", "Можешь быть уверен в этом",
    "Мне кажется — да", "Вероятнее всего",
    "Хорошие перспективы", "Да", "Знаки говорят — да",
    "Пока не ясно, попробуй снова", "Спроси позже",
    "Лучше не рассказывать тебе", "Сейчас нельзя предсказать",
    "Сконцентрируйся и спроси опять",
    "Даже не думай", "Мой ответ — нет",
    "По моим данным — нет", "Перспективы не очень хорошие",
    "Весьма сомнительно"
]
```

### Реализация — метод обратного преобразования

```python
def get_magic_8_ball():
    m   = len(MAGIC_ANSWERS)   # = 20
    p_i = 1.0 / m              # вероятность каждого ответа

    alpha = random.random()    # α ~ Uniform(0, 1)

    cumulative_p = 0.0
    for k in range(m):
        cumulative_p += p_i
        if alpha < cumulative_p:
            return MAGIC_ANSWERS[k]

    return MAGIC_ANSWERS[-1]
```

**Как работает метод:**

1. Генерируется число α из равномерного распределения U(0, 1).
2. Вычисляются кумулятивные вероятности: F(k) = (k + 1) / m.
3. Возвращается первый ответ с индексом k, для которого α < F(k).

Это эквивалентно разбиению отрезка [0, 1] на m равных частей и определению, в какую из них попало α:

```
[0 ──── 1/20 ──── 2/20 ──── ... ──── 19/20 ──── 1]
   k=0      k=1               k=18       k=19
```

Если α попало в интервал [(k−1)/m, k/m) — возвращается ответ с индексом k.
