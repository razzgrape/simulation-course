# Базовый датчик случайных чисел

---

## Математическая основа

### Равномерное распределение U[0, 1]

Оба генератора нацелены на воспроизведение равномерного распределения на отрезке `[0, 1)`. Его теоретические характеристики:

**Математическое ожидание:**
```
μ = (a + b) / 2 = (0 + 1) / 2 = 0.5000000
```

**Дисперсия:**
```
σ² = (b − a)² / 12 = 1 / 12 ≈ 0.0833333
```

Эти значения служат эталоном — чем ближе к ним выборочные оценки, тем качественнее генератор.

---

## Реализация

### LCG

Реализован классический ЛКГ. Генератор строит последовательность по рекуррентной формуле:

```
X_{n+1} = (a · X_n + c) mod m
```

Каждое следующее состояние однозначно определяется предыдущим — последовательность детерминирована и воспроизводима при одинаковом seed.

Из целочисленного состояния `X_n` получается вещественное число из `[0, 1)`:
```
u_n = X_n / m
```

**Начальные параметры**:

| Параметр | Значение        | Роль                                          |
|----------|-----------------|-----------------------------------------------|
| `m`      | 2³¹ = 2 147 483 648 | Модуль; определяет максимальный период    |
| `a`      | 1 103 515 245   | Множитель; |
| `c`      | 12 345          | Приращение; |
| `seed`   | 52              | Начальное состояние X₀                        |

---

### MT19937 

Для сравнения используется `random.random()` из стандартной библиотеки Python. 

Ключевые характеристики:

| Характеристика | Значение                    |
|----------------|-----------------------------|
| Период         | 2¹⁹⁹³⁷ − 1                 |
| Размерность равномерного распределения | 623 |
| Seed           | не фиксирован (системное время) |

Seed не фиксируется намеренно — каждый запуск даёт независимую последовательность, что соответствует поведению «реального» генератора в отличие от воспроизводимого ЛКГ.

---

### Вычисление статистик

Для каждой выборки вычисляются две оценки.

**Выборочное среднее** — несмещённая оценка математического ожидания:
```
x̄ = (1/n) · Σ xᵢ
```

**Дисперсия** :
```
s² = (1 / (n−1)) · Σ (xᵢ − x̄)²
```

## Результаты эксперимента (N = 100 000)

Результаты получены при запуске с параметрами по умолчанию: seed ЛКГ = 52, N = 100 000.

| Генератор         | Среднее x̄  | Δ среднее       | Погрешность μ | Дисперсия s²  | Δ дисперсия     | Погрешность σ² |
|-------------------|------------|-----------------|---------------|---------------|-----------------|----------------|
| Теоретические     | 0.5000000  | —               | —             | 0.0833333     | —               | —              |
| ЛКГ (собственный) | 0.4985401  | −1.46 × 10⁻³   | 0.2920 %      | 0.0838207     | +4.87 × 10⁻⁴   | 0.0975 %       |
| MT19937 (random)  | 0.4995476  | −4.52 × 10⁻⁴   | 0.0905 %      | 0.0833653     | +3.20 × 10⁻⁵   | 0.0064 %       |

Погрешность считается относительно теоретических значений:
```
δμ  = |x̄ − μ|  / μ  · 100 %
δσ² = |s² − σ²| / σ² · 100 %
```

## Вывод

При N = 100 000 оба генератора воспроизводят параметры равномерного распределения U[0, 1] в пределах ожидаемой статистической погрешности `~0.3%`.

MT19937 значительно точнее ЛКГ: погрешность по среднему в **3.2 раза** меньше (0.09% против 0.29%), по дисперсии — в **15 раз** меньше (0.006% против 0.098%). Это закономерно: MT19937 обладает несравнимо большим периодом и проходит значительно более строгие тесты случайности
